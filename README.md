# 实验室设备管理系统设备端

## 特性
* 二维码扫码跳转小程序登录使用
* 空闲、预约、使用三个界面
* 触控屏控制
* 使用了[LVGL](https://github.com/lvgl/lv_port_esp32)图形库
* 实现了微信头像显示
* 亿点点BUG
